Demos and Examples

On the following pages you'll find some examples of how to use PDL for basic computations and plotting purposes. Several of the examples are available as demos within perldl. For more details try:

 perldl> demo
  • pdl - Introduction -- a tour of PDL
  • bad - Bad-value support
  • cartography - Cartographic projections
  • dsp_windows - DSP windowing
  • func - Interpolation etc
  • gsl_cdf - GSL cumulative distribution functions
  • gsl_rng - GSL randomness functions
  • proj - Transforms with PROJ
  • simplex - Simplex optimisation
  • transform - Coordinate transformations
  • 2D plotting with PGPLOT
  • 3D plots
